# **App Name**: 蓝领快聘

## Core Features:

- 用工需求发布: 农民工可以在平台上发布用工需求，包括工种、工作地点、薪资待遇等信息。
- 职位信息展示: 清晰地展示所有的职位信息，包括工种、地点、薪资等。
- 职位搜索: 允许用户按工种、地区、薪资等条件搜索职位。
- 在线应聘: 工人可以在线提交应聘信息，包括个人简历、技能证书等。
- 自动职位匹配: 利用AI工具分析工人的技能和经验，自动匹配合适的职位。

## Style Guidelines:

- 主色调：蓝色 (#3498DB)，代表信任和专业。
- 背景色：淡蓝色 (#EBF5FB)，提供清晰和舒适的视觉体验。
- 强调色：橙色 (#F39C12)，用于突出关键按钮和信息。
- 正文和标题字体：'PT Sans'，兼具现代感和可读性。
- 使用简洁明了的图标，例如位置图标、薪资图标等，增强用户体验。
- 采用清晰的布局，信息分层展示，易于浏览和查找。
- 在页面切换和数据加载时加入平滑的过渡动画，提升用户体验。
